Paintball Field Builder Help

Registering The Software



Paintball Field Builder runs in the NON-COMMERCIAL mode until you register the software. The NON-COMMERCIAL mode disables saving, loading, and rendering.



To register the software click About->Register.




Fill out the information below and click Register. If it succeeds it will exit out of Paintball Field Builder, and the next time you start it will run in COMMECRIAL mode. If it fails to connect make sure that you are connected to the Internet, do not make more than 3 attempts within a minute. If it has an error of an invalid key, make sure you that you typed the key correctly.
