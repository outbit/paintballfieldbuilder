Paintball Field Builder Help

Saving And Loading


 

 

-Starting Fresh With a New Field-

To create a new field click File->New. Type in the dimensions of the field and yardline grid, then click ok. The scene will be cleared and the field and grid will be resized.


 

-Saving a Field-

To save a field that you have layed out, click File->Save or File->Save As.

 

-Loading a Field-

To load a field that you have layed out, click File->Open.

 

-Loading a Paintball Field Kit-

To load a field kit click File->Load Field Kit. Go the the fieldkits folder if its not already there. Then select any *.PFK file in that folder. To install a new field kit, just copy the file(*.PFK) to the field kit directory.
