Paintball Field Builder Help

Definitions

Bunker Package – Paintball Field Kit, a collection of bunker,objects,trees,etc.

OGRE – A very powerful Object Oriented Graphics Rendering Engine. Www.ogre3d.org

Perspective - A view that allows rotation around the scene.

First Person - A view that allows you to see the scene as if you were walking the field.

Top – A view that allows you to see the scene from the top view.

.PFK – A paintball field kit file also refered to as a bunker package.

Paintball Field Kit – Bunker Package, a collection of bunker,objects,trees,etc.

HPB – Hyperformance Paintball.

PFB – Paintball Field Builder.

Zoom – To move the camera closer and back.
