Paintball Field Builder v1.2.3 Help

Index



Paintball Field Builder is a 3D paintball field layout software. It features an easy to use interface for placing and manipulating bunkers in real-time. Placement grid allows you to easily design your field to scale for a more accurate results. Flexible design allows you to load custom bunker packages, this allows you to add new bunkers, netting, starting places, trees, etc that is not included in the default bunker package.



-New Features-

In First Person View You Can Use The Up And Down Arrows On The KeyBoard To Change The y Position Of The View (standing or kneeling).

The Trial Version Now Allows You To Open *.Field Files.
